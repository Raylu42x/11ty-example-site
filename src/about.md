# About

This is a minimal [Eleventy](https://www.11ty.dev/) starter site, designed to pair
with [11ty CMS](https://github.com/Raylu42x/11ty-cms).

It demonstrates:

- A site-wide `base.njk` layout and a `post.njk` post layout
- A `posts` collection sourced from `src/posts/*.md`
- Folder-level frontmatter via `posts.json`
- Site metadata in `src/_data/site.json`
- A passthrough-copied `assets/` directory
- An Atom feed at `/feed.xml`

Edit this file at `src/about.md` to make it your own.
